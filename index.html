Instructions

  • Enter an equation of a chemical reaction and click 'Balance'. The answer will appear below
  • Always use the upper case for the first character in the element name and the lower case for the second character. Examples: Fe, Au, Co, Br, C, O, N, F.
  • Compound states [like (s) (aq) or (g)] are not required and will not work
  • Please use either an equals sign, =, or an equals sign and a grater than sign, =>, for the yields sign between the reactants and the products
  • For equations with ions please use the ^ symbol like CO3^2- or NH4^+
  • For spare electrons use a single lower case e

Examples

  • Fe + Cl2 = FeCl3
  • KMnO4 + HCl => KCl + MnCl2 + H2O + Cl2
  • K4Fe(CN)6 + H2SO4 + H2O = K2SO4 + FeSO4 + (NH4)2SO4 + CO
  • C6H5COOH + O2 => CO2 + H2O
  • CO3^2- = O2 + CO2 + e
